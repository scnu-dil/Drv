Drv

Drv（Data-Driven value Coin)是一个面向教育领域的数据智能服务项目。DDEC致力于解决目前教育领域中缺乏数据共享，教学数据孤岛等问题中，通过教育物联网技术，实现教育的群体感知。DDEC通证是基于IoTex网络实现的生态项目，DDEC主要用于转账，奖励数据贡献者，例如知识图谱贡献者，教育采集者与社区参与者等。


### Env preparation

```

npm install

```
